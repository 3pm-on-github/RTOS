# RTOS proxy v0.2
### The RTOS proxy is a proxy used to navigate websites that aren't compatible with RTOS.
## How to install the RTOS proxy:
1. Download Node.js v22 [here](https://nodejs.org/).
2. Run the following command to install the dependencies: `npm install express node-fetch`
### You are done!
To run the RTOS proxy, Run the following command: `node proxy.js`
You will receive a link, that you will have to enter in the RTOS Browser.
